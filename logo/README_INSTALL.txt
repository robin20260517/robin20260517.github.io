安装方法：

1. 解压 canvas-design-robin.zip
2. 把整个 canvas-design-robin 文件夹复制到：
   C:\Users\你的用户名\.claude\skills\canvas-design-robin

Windows 也可以复制到：
   %USERPROFILE%\.claude\skills\canvas-design-robin

使用方式：

/canvas-design-robin
参考 https://f.io/ww64ZI6u 的质感和 https://yggdrasilgaming.com/games 的 iGaming 视觉方向，生成一个 DCC.bet 游戏 logo。要求：DCC 是主体，.bet 小一点，整体一行连贯，高饱和，3D 质感，财富游戏氛围，透明 PNG。

如果有参考图，把图片放进 references 文件夹，再这样说：

/canvas-design-robin
参考 references 文件夹里的图片，生成一个游戏 logo。
